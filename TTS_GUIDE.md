# TTS (Text-to-Speech) 기능 가이드

## 🔊 개요

브라우저 내장 TTS 기능을 사용하여 요약 내용을 음성으로 들을 수 있습니다.

## 📋 TTS 버튼 위치

### 1. 날짜별 TTS (🔊 날짜 듣기)
- **위치**: 각 날짜 헤더 우측
- **기능**: 해당 날짜의 모든 영상 요약을 순서대로 읽어줌
- **형식**: "제목: [영상제목]. 요약: [요약내용]" 반복

```
2026-02-12 (목)  [🔊 날짜 듣기] ▼
```

**사용 예시**:
- 출퇴근 시간에 오늘의 모든 요약 듣기
- 운동하면서 전체 내용 파악

### 2. 채널별 TTS (🔊 채널 듣기)
- **위치**: 각 채널 헤더 우측
- **기능**: 해당 채널의 모든 영상 요약을 순서대로 읽어줌
- **형식**: "제목: [영상제목]. 요약: [요약내용]" 반복

```
📺 삼프로TV_경제의신과함께  [🔊 채널 듣기] ▼
```

**사용 예시**:
- 특정 채널만 집중해서 듣기
- 좋아하는 채널의 모든 업데이트 확인

### 3. 영상별 TTS (🔊 요약 듣기)
- **위치**: 각 영상 내용 상단
- **기능**: 해당 영상의 요약만 읽어줌
- **형식**: 요약 내용만

```
게시일: 2026-02-12 오전 9:00:00
[🔊 요약 듣기]
[요약 내용]
```

**사용 예시**:
- 관심 있는 영상만 선택해서 듣기
- 긴 요약을 읽기 대신 듣기

## 🎮 사용 방법

### 기본 사용

1. **재생**: 🔊 버튼 클릭
2. **중지**: ⏹ 버튼 클릭 (재생 중일 때)
3. **다른 TTS 재생**: 자동으로 이전 TTS 중지 후 새로운 TTS 재생

### 버튼 상태

| 상태 | 아이콘 | 색상 | 설명 |
|------|--------|------|------|
| 대기 | 🔊 듣기 | 회색 | 클릭하면 재생 시작 |
| 재생 중 | ⏹ 중지 | 빨강 | 클릭하면 중지 |

### 단축키 (브라우저 기본)

대부분의 브라우저에서 TTS 중지:
- **Esc** 키 (일부 브라우저)
- 페이지 새로고침

## ⚙️ 음성 설정

### 음성 속도 및 음높이

현재 기본 설정:
- **속도 (rate)**: 1.0 (보통)
- **음높이 (pitch)**: 1.0 (보통)
- **언어**: ko-KR (한국어)

### 커스터마이징 (개발자용)

`src/htmlTemplate.ts` 파일에서 수정:

```typescript
const utterance = new SpeechSynthesisUtterance(text);
utterance.lang = 'ko-KR';
utterance.rate = 1.2;  // 1.2배속 (0.1 ~ 10)
utterance.pitch = 1.0; // 음높이 (0 ~ 2)
utterance.volume = 1.0; // 볼륨 (0 ~ 1)
```

## 💡 사용 팁

### 1. 멀티태스킹

```
운전 중 → 날짜별 TTS로 전체 듣기
요리 중 → 채널별 TTS로 관심 채널만
운동 중 → 영상별 TTS로 선택해서 듣기
```

### 2. 효율적인 정보 습득

```
1단계: 날짜별 TTS로 전체 훑기
2단계: 관심 있는 채널만 채널별 TTS
3단계: 중요한 영상만 영상별 TTS + 원문 읽기
```

### 3. 시간 절약

```
출퇴근 30분 = 약 10-15개 요약 청취 가능
점심시간 10분 = 약 3-5개 요약 청취 가능
```

## 🔧 문제 해결

### TTS가 작동하지 않음

**증상**: 버튼 클릭해도 소리 없음

**해결 방법**:
1. 브라우저 음소거 확인
2. 시스템 볼륨 확인
3. 다른 탭에서 미디어 재생 중지
4. 브라우저 재시작
5. 다른 브라우저 시도

### TTS가 중간에 멈춤

**증상**: 재생 중 갑자기 중지

**원인**:
- 브라우저 메모리 부족
- 긴 텍스트 (5분 이상)
- 백그라운드 탭 제한

**해결 방법**:
1. 영상별 TTS 사용 (짧은 단위)
2. 다른 탭 닫기
3. 브라우저 재시작

### 음성 품질이 나쁨

**증상**: 로봇 같은 음성, 부자연스러움

**해결 방법**:
1. Chrome 또는 Edge 사용 (최고 품질)
2. 시스템 TTS 음성 업데이트
   - **Windows**: 설정 → 시간 및 언어 → 음성
   - **macOS**: 시스템 환경설정 → 손쉬운 사용 → 음성
   - **Android**: 설정 → 접근성 → TTS 출력

### "이 브라우저는 TTS를 지원하지 않습니다"

**해결 방법**:
1. 브라우저 업데이트
2. 지원 브라우저로 변경 (Chrome, Edge, Safari)

## 🎯 활용 시나리오

### 시나리오 1: 아침 출근길 (30분)

```
1. 날짜별 TTS로 오늘 전체 듣기 (15분)
2. 관심 채널만 다시 듣기 (10분)

```


```
1. 날짜별 TTS로 오늘 + 어제 (30분)
2. 관심 채널 전체 (30분)
```

### 시나리오 4: 집안일 중 (20분)

```
1. 채널별 TTS로 좋아하는 채널 2-3개 (20분)
```
