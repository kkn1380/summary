# 사용자 가이드

## 🎯 주요 기능

이 사이트는 YouTube 투자 채널의 영상을 AI로 요약하여 제공합니다.

### 핵심 컨셉
- **요약 중심**: 영상을 보지 않아도 핵심 내용 파악
- **계층 구조**: 날짜 → 채널 → 영상 순서로 탐색
- **선택적 로딩**: 필요한 내용만 펼쳐서 보기

## 📱 화면 구성

### 상단
```
투자 인사이트 | Investment Insights
YouTube 채널 요약 모음

📺 구독 중인 채널
[삼프로TV] [김작가 TV] [신사임당] ...
```

### 메인 콘텐츠
```
📅 오늘 (2026-02-11)          9개 영상 ▼
  📺 삼프로TV                  5개 영상 ▶
  📺 김작가 TV                 3개 영상 ▶
  📺 신사임당                  1개 영상 ▶

📅 어제 (2026-02-10)          ?개 영상 ▶
📅 2026-02-09 (일)            ?개 영상 ▶
...
```

## 🖱️ 사용 방법

### 1단계: 날짜 선택
```
📅 오늘 (2026-02-11) ▼  ← 클릭하면 접기/펼치기
```

- **오늘**: 자동으로 펼쳐진 상태
- **과거**: 클릭하면 데이터 로딩 후 표시
- **파란색**: 펼쳐진 상태
- **회색**: 접힌 상태

### 2단계: 채널 선택
```
📺 삼프로TV    5개 영상 ▶  ← 클릭하면 영상 목록 표시
```

- 관심 있는 채널 클릭
- 영상 목록이 표시됨
- 다시 클릭하면 접기

### 3단계: 영상 선택
```
연준의 금리 동결, 하지만...  [▶] ▶  ← 제목 클릭: 요약 보기
                                    ← [▶] 클릭: YouTube 이동
```

#### 제목 클릭
- 요약 내용 표시/숨김
- 여러 영상 동시에 펼칠 수 있음
- 다시 클릭하면 접기

#### YouTube 아이콘 [▶] 클릭
- 새 탭에서 YouTube 영상 열기
- 요약은 펼쳐지지 않음
- 바로 영상 시청 가능

## 💡 사용 팁

### 빠르게 오늘 요약 보기
1. 페이지 접속 (오늘 날짜 자동 펼침)
2. 관심 채널 클릭
3. 영상 제목 클릭
4. 요약 읽기

### 여러 영상 비교하기
1. 채널 펼치기
2. 첫 번째 영상 클릭 → 요약 확인
3. 두 번째 영상 클릭 → 요약 확인
4. 스크롤하며 비교
5. 관심 없는 영상 클릭 → 접기

### 특정 날짜 찾기
1. 스크롤하여 날짜 찾기
2. 날짜 클릭 (과거는 로딩)
3. 채널 펼치기
4. 영상 확인

### 바로 영상 시청
1. 채널 펼치기
2. 제목만 보고 판단
3. YouTube 아이콘 [▶] 클릭
4. 바로 시청

## 🎨 색상 의미

### 날짜 헤더
- 🔵 **파란색**: 펼쳐진 상태 (내용 표시 중)
- ⚪ **회색**: 접힌 상태 (클릭하여 펼치기)

### 채널 헤더
- 📺 **연한 회색**: 기본 상태
- 🔵 **파란색 테두리**: 채널 구분

### 영상 헤더
- 📄 **흰색**: 기본 상태
- 🔴 **빨간색 테두리**: 영상 구분
- 🔴 **빨간색 버튼**: YouTube 링크

## 📊 데이터 로딩

### 즉시 표시 (빠름)
- 오늘 날짜의 모든 데이터
- 이미 펼친 날짜의 데이터

### 클릭 시 로딩 (약간 느림)
- 과거 날짜 (처음 클릭 시)
- "데이터를 불러오는 중..." 표시
- 한 번 로드하면 캐시됨

## 🔍 검색 팁

### 특정 채널 찾기
1. 상단 "구독 중인 채널" 확인
2. 날짜 펼치기
3. 채널 이름으로 찾기

### 특정 주제 찾기
1. 여러 날짜 펼치기
2. 채널별로 확인
3. 제목에서 키워드 찾기
4. 요약 펼쳐서 확인

### 최신 정보 확인
1. 오늘 날짜 (자동 펼침)
2. 모든 채널 빠르게 훑기
3. 관심 영상만 펼치기

## 📱 모바일 사용

### 터치 동작
- **탭**: 펼치기/접기
- **스크롤**: 날짜 탐색
- **길게 누르기**: 없음 (일반 탭만)

### 최적화
- 작은 화면에 맞춰 조정됨
- 터치하기 쉬운 크기
- 스크롤 최소화

## ⚡ 성능

### 빠른 로딩
- 초기: 오늘 데이터만 (약 10개 영상)
- 과거: 클릭 시 로딩
- 캐시: 한 번 로드하면 재사용

### 메모리 절약
- 접힌 내용은 렌더링 안 함
- 필요한 것만 표시
- 부드러운 스크롤

## 🆘 문제 해결

### "데이터를 불러오는데 실패했습니다"
- 새로고침 (Ctrl+R 또는 Cmd+R)
- 인터넷 연결 확인
- 잠시 후 다시 시도

### 날짜가 로딩되지 않음
- 해당 날짜에 데이터가 없을 수 있음
- 다른 날짜 시도
- 새로고침

### 요약이 표시되지 않음
- 영상 제목 클릭 확인
- YouTube 아이콘이 아닌 제목 클릭
- 다시 클릭하여 펼치기

## 🎯 효율적인 사용법

### 아침 루틴
1. 페이지 접속

4. 관심 영상 모아서 시청

1. 여러 날짜 펼치기
2. 키워드로 제목 검색
3. 관련 영상 요약 비교
4. 가장 유용한 영상 시청
