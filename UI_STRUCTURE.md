# UI 구조 설명

## 📊 3단계 계층 구조

```
📅 날짜 (2026-02-11) ▼                    ← 클릭: 날짜 펼침/접기
  │
  ├─ 📺 삼프로TV (5개 영상) ▼              ← 클릭: 채널 펼침/접기
  │   │
  │   ├─ 📄 연준의 금리 동결... ▼          ← 클릭: 요약 펼침/접기
  │   │   └─ [요약 내용 표시]
  │   │
  │   ├─ 📄 코스피가 1만 간다고... ▼
  │   │   └─ [요약 내용 표시]
  │   │
  │   └─ ...
  │
  ├─ 📺 김작가 TV (3개 영상) ▼
  │   │
  │   ├─ 📄 비트코인 전망... ▼
  │   │   └─ [요약 내용 표시]
  │   │
  │   └─ ...
  │
  └─ ...

📅 날짜 (2026-02-10) ▶                    ← 접힌 상태 (클릭 시 로딩)
📅 날짜 (2026-02-09) ▶
...
```

## 🖱️ 사용자 인터랙션

### 초기 로딩
1. 페이지 로드
2. `index.json` 읽기 (오늘 데이터 포함)
3. 오늘 날짜만 펼쳐진 상태로 표시
4. 오늘의 모든 채널은 접힌 상태

### 날짜 클릭
```
접힌 날짜 클릭
  ↓
해당 날짜 JSON 로드 (최초 1회)
  ↓
채널 목록 표시 (모두 접힌 상태)
```

### 채널 클릭
```
접힌 채널 클릭
  ↓
영상 목록 표시 (모두 접힌 상태)
```

### 영상 클릭
```
영상 제목 클릭
  ↓
요약 내용 표시/숨김 (토글)
```

### YouTube 아이콘 클릭
```
[▶] 아이콘 클릭
  ↓
YouTube 새 탭에서 열기
  ↓
요약은 펼쳐지지 않음 (event.stopPropagation)
```

## 📱 반응형 디자인

### 데스크톱 (1200px+)
- 최대 너비: 1200px
- 여백: 30px
- 폰트: 기본 크기

### 태블릿 (768px ~ 1199px)
- 최대 너비: 100%
- 여백: 20px
- 폰트: 약간 작게

### 모바일 (~ 767px)
- 최대 너비: 100%
- 여백: 15px
- 폰트: 작게
- 채널 태그: 작게

## 🎯 데이터 로딩 전략

### 즉시 로딩 (페이지 로드 시)
- `index.json` (메타데이터 + 오늘 데이터)
- 오늘 날짜의 모든 영상 요약

### 지연 로딩 (클릭 시)
- 과거 날짜의 JSON 파일
- 한 번 로드하면 캐시 (재클릭 시 즉시 표시)

### 로딩 상태 표시
```
날짜 클릭 → "데이터를 불러오는 중..." 표시
         ↓
JSON 로드 완료 → 채널 목록 표시
```

### 토글 아이콘

- 접힘: 90도 회전 (▶)
- 펼침: 0도 (▼)

### 호버 효과
- 배경색 변화 (부드럽게)
- 커서: pointer

## 📊 성능 최적화

### 초기 로딩
- 오늘 데이터만 (약 10개 영상)
- 나머지는 접힌 상태 (렌더링 안 함)

### 메모리 사용
- 로드된 날짜 추적 (`loadedDates` Set)
- 중복 로딩 방지

### 네트워크
- 필요한 JSON만 로드
- 한 번 로드하면 재사용

## 🆚 기존 vs 새로운 구조

### 기존 (정적)
```
모든 날짜
  └─ 모든 채널
      └─ 모든 영상
          └─ 모든 요약 (항상 표시)
```
- 초기 로딩: 느림 (모든 데이터)
- 스크롤: 길음
- 찾기: 어려움

### 새로운 (동적)
```
오늘 (펼침)
  └─ 채널 (접힘) → 클릭
      └─ 영상 (접힘) → 클릭
          └─ 요약 (표시)

과거 (접힘) → 클릭 → 로드
```
- 초기 로딩: 빠름 (오늘만)
- 스크롤: 짧음
- 찾기: 쉬움 (계층 구조)

## 🎯 사용 시나리오

### 시나리오 1: 오늘 요약 빠르게 보기
1. 페이지 접속
2. 오늘 날짜 이미 펼쳐짐
3. 관심 채널 클릭
4. 관심 영상 제목 클릭
5. 요약 읽기
6. (선택) YouTube 아이콘 클릭 → 영상 시청

### 시나리오 2: 여러 영상 요약 비교
1. 채널 펼치기
2. 첫 번째 영상 클릭 → 요약 읽기
3. 두 번째 영상 클릭 → 요약 읽기
4. 첫 번째 영상 다시 클릭 → 요약 숨김
5. 관심 있는 영상만 YouTube 아이콘 클릭

### 시나리오 3: 특정 영상 바로 시청
1. 채널 펼치기
2. 영상 제목 확인
3. YouTube 아이콘 클릭 (요약 안 봄)
4. 바로 영상 시청

완벽한 3단계 계층 구조! 🎉
